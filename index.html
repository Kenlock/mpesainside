MPESA Integration now simple free open-source cross-platform

This project, borne from a developers' meetup at iHub, seeks to provide a free and standard method of integrating MPESA payments in custom projects.

Free

The code is free to download and use, and there are no charges for integrating this in a custom system

Simple

No lengthy setup processs, just grab a copy of the code, and defining a couple of constants and you are good to go.

Open-Source

This is a collaborative project by a number of local and foreign devs. Feels free to contribute to the code here on Github.

Cross-platform

We currently support the following scripting languages: PHP, Python, JavaScript, NodeJS, Java/Kotlin and Ruby.
